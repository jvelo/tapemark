# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project adheres
to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Changed

- Relicensed from MIT to **MPL-2.0**. Every source file now carries an SPDX/MPL
  header, enforced via ESLint (`pnpm run lint`).
- Packages now publish to the public npm registry under `@jvelo/*`.

### Added

- Tests for the Cloudflare D1 adapter.
- Per-package READMEs, license files, and npm metadata.
